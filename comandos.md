
Para rodar o arquivo corretamente, você precisa executar de dentro da pasta backend:

cd Alessandra-trabalho-\backend
node server.js




Ou diretamente com o caminho:

node .\Alessandra-trabalho-\backend\whatsapp-qr.js



cd .\painel
npm install          # instala as dependências (executa apenas na primeira vez)
npm start             # inicia o servidor de desenvolvimento (geralmente em http://localhost:3000)
